## What is CSS: CSS is a class based stylesheet language.

## Basic syntex of CSS:
    selector{
        property: value;
    }

## Three types of CSS selectors:
    1. HTML tags
    2. User defined name
    3. Special syntex

## Block level and Inline Elements: Block level elements grab the full row. But inline elements doesn't grab
   the full row. It grabs the only the area which consume it's contents.
